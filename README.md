# AI Interview Assistant

AI-powered chatbot using MERN & Gemini AI to help users prepare for technical and HR interviews.

## Setup

### Backend (Node.js & Express)
1. `cd backend`
2. `npm install`
3. Add a `.env` file with `PORT=5000`, `MONGO_URI=your_mongodb_uri`, and `GEMINI_API_KEY=your_gemini_key`
4. `npm run dev` (starts on port 5000)

### Frontend (React & Vite)
1. `cd frontend`
2. `npm install`
3. `npm run dev`
